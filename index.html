The platform streamlines incident management, supports timely intervention, and helps foster a safer school environment.

  • MySQL
  • CSS
  • Bootstrap

Key Features

  • Anonymous and confidential incident reporting
  • Incident tracking and status updates

SpecSnap

Project Overview

SpecSnap is a tool designed to quickly generate QR codes containing detailed hardware specifications for devices such as computers, laptops, and mobile devices. It streamlines the process of sharing and accessing hardware information, making it ideal for IT departments, tech support, and inventory management.

Technologies Used

  • PHP
  • MySQL
  • CSS
  • Bootstrap
  • QR Code API

Key Features

  • Easy input of hardware specifications
  • Instant QR code generation for each device
  • Downloadable and printable QR codes
  • Mobile-friendly interface
  • Search and filter devices by specification
